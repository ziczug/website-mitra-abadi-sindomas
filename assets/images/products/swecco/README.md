- Resolusi disarankan: **400×400 px** atau lebih (ratio 1:1)
- Nama file **HARUS PERSIS** seperti kolom "Nama File" di atas
- Setelah upload, jalankan `node sync_cms.js` untuk sinkronkan ke dashboard
